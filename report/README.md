
# 操作系统作业调度算法实验报告

## 项目简介
本项目实现了三种经典的作业调度算法：先来先服务（FCFS）、短作业优先（SJF）和最高响应比优先（HRRN），并在单处理器和双处理器环境下进行了性能对比分析。

## 实验内容
- **算法实现**：FCFS、SJF、HRRN三种非抢占式调度算法
- **环境测试**：单处理器和双处理器环境下的性能对比
- **数据分析**：周转时间、带权周转时间等关键指标的统计分析
- **性能评估**：不同算法在不同任务流模式下的表现分析

## 技术特点
- **编程语言**：Rust（版本1.70+）
- **设计模式**：面向对象设计，模块化架构
- **数据结构**：Task结构体，支持多处理器调度
- **算法优化**：事件驱动调度，高效的时间管理

## 实验结果
- **单处理器环境**：SJF算法表现最优，平均周转时间10.9000分钟
- **双处理器环境**：三种算法性能一致，平均周转时间6.0000分钟
- **性能提升**：双处理器相比单处理器，平均周转时间降低50%以上
